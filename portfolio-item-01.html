YNF Media Player

A Modern & Dynamic Player

It is a Visual C# project developed by me & my two classmates in undergrad. Almost all of the portions of this project have been developed by me.

YNF Media Player is a dynamic media player written in C# with the help of the .NET framework in Visual Studio. It can play media files as well as can stream and download youtube videos.

It also has a login system that uses Microsoft SQL Server Database. I have done the whole project actually, and others are with me for academic marks. Bunifu framework and Modern UI C# have been used in this project too.
